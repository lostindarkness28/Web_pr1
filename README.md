# Приморська ВЕС — Енергія Вітру

Адаптивний лендінг про об’єкт відновлюваної енергетики в Запорізькій області. Проєкт реалізовано з використанням сучасних стандартів веб-розробки.

## Технології
* **HTML5** — структура.
* **CSS3** — стилізація та власні змінні.
* **Bootstrap 5** — адаптивна сітка та компоненти.
* **Figma** — макет та дизайн.

## Як запустити
1. Переконайтеся, що файл `index.html` та папки `css/` і `images/` знаходяться в одній директорії.
2. Відкрийте файл **index.html** у будь-якому браузері (Chrome, Firefox, Edge).
3. Для розробки рекомендується використовувати розширення **Live Server** у VS Code для автоматичного оновлення сторінки при зміні коду.

## Функціонал та особливості
* **Адаптивність:** сторінка автоматично підлаштовується під будь-яку ширину екрана (смартфону, планшети, ПК).
* **Валідація:** код пройшов перевірку W3C на відсутність помилок.
* **Інформативність:** наведено технічні параметри (200 МВт), схему об'єкта та опис інфраструктури.
